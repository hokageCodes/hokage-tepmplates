# Modern Launch Templates – Canonical Landing Page

A clean, neutral, typography-led landing page for the Modern Launch Templates bundle.

## Purpose
- Explain the product
- Show what’s included
- Send users to download the templates

## Stack
- Next.js (App Router)
- Tailwind CSS
- Static site only

## Design System
- Inter font
- Neutral palette
- Indigo/blue accent
- No gradients or illustrations

## Structure
- Single page
- No backend, CMS, auth, or forms

## Sections
1. Hero
2. Template Grid
3. What You Get
4. Who It’s For / Not For
5. FAQ
6. Final CTA
7. Footer
